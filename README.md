# Software-Engineering-A2

Este repositório apresenta o trabalho de A2 da disciplina Engenharia de Software ministrada pelo professor doutor Rafael Pinho.

### Sobre o trabalho

A descrição completa do trabalho se encontra no arquivo `ProjetoIndividualModelagem.pdf`. 
O intuito é colocar em prática, ainda que num contexto limitado, os conhecimentos de padrões de projeto adquiridos ao longo do semestre.
Uma vez que gosto de temas relacionados à saúde, decidi fazer a modelagem de uma árvore de decisão fictícia (mock) para um processo de **triagem de emergência**.
